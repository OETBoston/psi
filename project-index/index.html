Seat Light Control

LightWell

By: Michelle Laboy, Joshua Fiedler, Seth Wiseman

Solar-powered public seat and planter that visualizes Boston's stormwater systems through LED lights connected to real-time groundwater sensors

LightWell v2*

By: Michelle Laboy, Joshua Fiedler, Seth Wiseman, Boston Groundwater Trust

Building off of the original LightWell project, this is a citizen science project using sensor-connected well caps and a mobile app to continuously collect and visualize groundwater data citywide

Tidraphone

By: Kate Balug, Dana Busch, Alyson Fletcher, Emily Goldsmith, Conor MacDonald, Joseph Wight, Noah Vawter

Interactive tidal vibraphone installed on Congress Street Bridge that plays different melodies based on tide levels, using sound to engage the public with climate change and sea level rise

Stairs of Fabulousness

By: Liz LaManche

Temporary rainbow staircase installation at Boston City Hall using fluorescent grit tape to improve safety, visibility, and fabulousness

Portable Reading Room

By: Leslie Davol, Sam Davol, Höweler + Yoon Architecture, Diana Limbach Lempel

Mobile library on the Rose Kennedy Greenway
